Other Projects

WavyColors.JS Open Source

In order to gain more control over the use of color on web pages, I built a light-weight, easy-to-use JS library that allows a programmer to type color classes directly into HTML code. Notice the color changes in this text? If you refresh the page, it will change slightly every time which adds interest to the user experience.

View on Github

RandomColors.JS Open Source

This JS library allows text and other objects to change colors in systematic, but stochastic ways. In other words, the colors of objects change every time a page is loaded within a range. For example, the grey used in the text you're reading now. Refresh this page to check it out! This is a work in progress, so feel free to contribute :-)
